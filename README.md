# SmartShares

Short Shareholder Management  API using Spring boot



### Setting up

- Ensure you have the following installed on your PC:

    - docker
    - docker-compose

## Getting Started

- Using Maven
```cmd
> mvnw clean spring-boot:run
```
- Using Docker
```cmd
> docker  compose up -d
```

### Assignment of Status codes

- `GET :: 200`
- `POST :: 202`
- `PUT :: 200`
- `DELETE :: 202`
