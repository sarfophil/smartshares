# SmartShares

Short Shareholder Management  API using Spring boot



### Setting up

- Ensure you have the following installed on your PC:

    - docker
    - docker-compose

## Getting Started

- Using Maven
```bash
> mvnw clean spring-boot:run
```
-Using Docker
```bash
> docker  compose up -d
```
Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.


### Frameworks

- Spring Framework
